# Family Chore Tracker - Deployment Guide

## Quick Deploy to Vercel (Recommended - 5 minutes!)

### Step 1: Download this folder
Download all the files in the `chore-tracker` folder to your computer.

### Step 2: Create a GitHub account (if you don't have one)
1. Go to https://github.com
2. Click "Sign up" and create a free account

### Step 3: Upload to GitHub
1. Go to https://github.com/new
2. Name your repository: `family-chore-tracker`
3. Keep it Public
4. Click "Create repository"
5. Click "uploading an existing file"
6. Drag and drop ALL the files from the chore-tracker folder
7. Click "Commit changes"

### Step 4: Deploy to Vercel
1. Go to https://vercel.com
2. Click "Sign Up" and choose "Continue with GitHub"
3. Click "Add New Project"
4. Find your `family-chore-tracker` repository and click "Import"
5. Vercel will auto-detect it's a Vite project
6. Click "Deploy"
7. Wait 2-3 minutes for deployment to complete

### Step 5: Get Your URL!
Once deployed, Vercel will give you a URL like:
`https://family-chore-tracker-xxxxx.vercel.app`

Save this URL and add it to your phone's home screen!

## Add to Phone Home Screen

### iPhone:
1. Open the URL in Safari
2. Tap the Share button
3. Scroll down and tap "Add to Home Screen"
4. Name it "Chore Tracker"
5. Tap "Add"

### Android:
1. Open the URL in Chrome
2. Tap the three dots menu
3. Tap "Add to Home screen"
4. Name it "Chore Tracker"
5. Tap "Add"

## Features

- ✅ Two caregiver accounts
- ✅ Multiple child profiles with age-appropriate tasks
- ✅ Points system with levels
- ✅ Streak tracking with bonus points
- ✅ Rewards shop
- ✅ Separate kid-friendly interface
- ✅ Persistent data storage

## Customization

Once deployed, you can:
- Add/edit caregivers during initial setup
- Add each child with their age (tasks auto-adjust)
- Kids can track their own progress
- Caregivers can award points throughout the day

## Troubleshooting

If the app doesn't load:
- Clear your browser cache
- Make sure all files were uploaded to GitHub
- Check Vercel deployment logs for errors

## Support

Need help? The app stores all data in browser storage, so it works offline once loaded!
